CMP9785M Workshop 1- Development Environment Setup

 

This workshop is in three parts that cover setting up access to the Azure cloud platform and configuration of your development environment:

Part 1 - Setting up your subscription

Part 2 - Setup development environment

Part 3 - Test your Azure for Students subscription

 

Part 1 - Setting up your subscription

In order to complete the module workshops and assessment you will need an azure subscription which should have been provided to you. If you have not been provided a subscription, please email dfoster@lincoln.ac.uk to request one.

When you have completed the registration for your subscription, navigate to https://portal.azure.com , and sign in using your credentials. Next, check your subscription is active by selecting the menu on the left by clicking the 'three lines' icon on the top left. From the menu that pops out select the 'Subscriptions' option and check you have an entry for the subscription. If you are having trouble activating your subscription please get help from the module team as soon as possible, it is your responsibility to activate your subscription, and it is mandatory to complete the assessment and carry out a large number of workshop tasks.

Part 2 - Setting up the development environment

The main development tool you will use is Visual Studio Code (VSC) with the required extensions for connecting to, and developing cloud services on Azure. This guide walks you through the main steps of installing and configuring the required frameworks and VSC for use in the module, please do not use 'Visual Studio', this is a completely different tool and not what we will be using in the module.

Frameworks & Tools

Frameworks

The main frameworks required are Microsoft .NET SDK Core 6 (x64) and Microsoft .NET SDK Core 3.1.x - we MIGHT need both versions for backwards compatability with some VSC extensions and libraries. Please download and install them for your Operating System, remember Windows is the recommended OS as this is what we support on our Lab machines.

Tools

We need several Azure supporting tools to aid your development work. We may add more as the module progresses but for now you should install the Azure Functions Core Tools package and the Azure Storage Explorer and the Azure IOT Explorer: Download and install the Azure Functions Core Tools here and the Azure Storage Explorer here. Also the Azure IOT Explorer here.
